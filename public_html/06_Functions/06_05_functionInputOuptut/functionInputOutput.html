Function Input

A function which adds two numbers provided as input.

How it Works:

This program uses a function to add 2 numbers and is an example of a function with input and output. The numbers are created as local variables in setup() and passed to the function as arguments. The function stores the input ‘args’ as local variables, which it adds together. The function returns the sum of the numbers. The value returned from the function is printed out from within setup().
