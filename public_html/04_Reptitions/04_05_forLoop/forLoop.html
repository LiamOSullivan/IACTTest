For Loop

Uses a loop to print one hundred statements to the console.

How it Works:

The count variable is appended to the text line with the line number (from 1 to 100).
The print statement prints out the value of the count variable at each loop iteration.
We have declared count within the loop header, but within the loop it is treated
as any other variable.
We can access it, but be careful that it is not modified (e.g. incremented) such that the loop works in an unexpected way.
