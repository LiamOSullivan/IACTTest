Moving Circle

Animating a shape with motion.

How it Works:

We need the variable for the x-position to have global scope,so that it may be modified from within the setup() and draw() functions. Therefore, we declare it at the top of the sketch.
Redrawing the background means that the previous circle will be erased before the new one is drawn.
Comment out this statement to see the effect of not redrawing the background.
circleX += 1; increments the variable by 1 every frame, so the circle moves 1 pixel at a time.
circleX += speed; increments the variable by the value stored in the speed variable.
