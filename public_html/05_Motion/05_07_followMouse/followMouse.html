Follow Mouse

A circle follows the mouse around the canvas.

How it Works:

We can use the mouse position to control the value of a variable or any other parameter we desire.
This uses mouseX and mouseY to update the position of a circle onscreen.
As these system variables are continuously updated, the circle follows the mouse cursor around the display.
The system variables pmouseX and pmouseY always contain the position of the mouse in the frame previous to the current frame.
