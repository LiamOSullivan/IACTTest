Section 2, Lesson 2

Variable Data

How it Works:

Variables are declared using the var keyword. Values are written to a variable using the assignment operator, which looks like an equals sign; =

Each statement in this code sketch does the following:
Declares a var named count, assigns it the value 0.
Declares a var named letter, assigns it the value 'a'.
Declares a var named d, assigns the it value 132.32.
Declares a var named happy, assigns it the value false.
Declares a var named x, assigns it the value 4.0.
We can read the values stored in the variables by referring to their names inside println().
