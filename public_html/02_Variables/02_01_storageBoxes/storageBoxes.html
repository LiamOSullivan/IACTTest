A variable is like a box for storing data.
A variable is first declared before being used.
This effectively 'creates the box'.


Consider a box that holds a whole number.

We can:
  • Read from the box, copying the number.
  • Write to the box, replacing the number.

For useful computer programs, we will need many boxes to store lots of values.
We need to identify each box in our code in a way that makes it easy to remember what value they hold.

Storage Boxes

Shows how variables may be understood as 'storage boxes' in computer memory which hold values.

How it Works:

Two 'boxes' are created, these are variables called boxA and boxB.
Two numbers are placed into the boxes- a write operation.
The numbers are printed out by reading from the boxes.
New numbers are then written to the boxes, and these are read out and printed.
Finally, a number from one box is copied to the other using a combined read and write.
The values in the boxes are again printed out.
