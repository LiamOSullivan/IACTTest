Colouring In Shapes

Shapes can be filled with a colour.
Here, a black square is drawn on a white canvas background.
A grey circle is also drawn.
Finally, a rectangle is drawn with no fill colour, so that the background color can be seen.

How it works:

The canvas is coloured white using background(). The number for white is 255
The colour used to paint a shape is set using the fill() function.
You can think of this as putting your paintbrush in a particular paint colour, before starting to paint the shape.
So fill() is called BEFORE the shape is drawn.
Like background(), colour is set using a number as an argument. So to colour the square black, fill(0) is used.
The rect() function is used to draw squares and rectangles, its arguments work like ellipse(): x, y, width, height.
After drawing the square and before drawing the circle, the fill colour must be changed to grey. Think of dipping your paintbrush in a new colour.
If the number 0 is black and 255 is white, grey is some number in-between. Try putting 125 in the second fill() function.
The circle is then drawn with the fill colour grey. to draw a shape that is transparent (has no colour), you can use noFill(). It's like cleaning your paintbrush!
A rectangle can be drawn using rect(), this time by setting different values for width and height.
