Hello World

Prints the common 'Hello World' phrase to the console window.

How it works:

This sketch has multiple statements. Each ends in a semicolon: ;.
Statements are executed in order, from top to bottom.
Each statement here contains a function call, which carries out a particular task.
Functions are recognizable as the function name is followed by a set of parentheses (regular brackets).
Arguments are provided to functions within the brackets, to tell the function how to carry out their tasks.
If a function takes multiple arguments, they are separated by a comma.

In this case, the message to be printed out is surrounded in double quotes
This says that the message should be printed exactly as it appears; it is a literal.
