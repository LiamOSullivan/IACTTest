Colouring Outlines

Outlines can be drawn around shapes, of varying colour and thickness.
Here, a white square with a black outline is drawn on a grey canvas.
A thick white circle outline with no fill colour is also drawn..
Finally, a white rectangle is drawn with no outline.

How it works:

The canvas is coloured grey this time.
The outline colour is set to black using the stroke() function. Just like fill() the number supplied sets the colour.
(The default outline colour is black, so if stroke() is not called, this will be used).
The thickness of the outline is set by strokeWeight() which again must be called BEFORE the shape is drawn.
Think of it as choosing a fatter or thinner paint brush.
The order of stroke() and strokeWeight() don't matter, once they are both called before the shape is drawn.
After drawing the square with a black outline, noFill() is used before the circle, the outline colour is set to white, and the stroke weight is increased.
Finally, the rectangle is drawn using noStroke(), so it appears without an outline.
