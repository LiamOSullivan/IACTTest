Printing Messages

More complex messages can be printed to the browser console.

How it works:

Short messages can be added together by enclosing them in double quotes and combining them with the concatenation operator: +
Messages can be formatted using special characters.
For example, a tab space can be inserted using a slash followed by the letter 't': \t
The print cursor can be moved down a line using the newline sequence, a slash followed by the letter 'n': \n
