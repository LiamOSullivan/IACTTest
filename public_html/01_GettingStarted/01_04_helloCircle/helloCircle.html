Hello Circle

Draws a circle on a canvas element.

How it works:

A canvas is an element which can be drawn to and is created using the createCanvas() function.
The width and height of the canvas are put within the brackets of the function, separated by a comma.
Sizes are measured in units of screen pixels.

The background() function sets the canvas colour.
Providing the number 0 as an argument sets the background colour black.

The ellipse() function is used to draw a circle or ellipse.
It takes 4 arguments within its brackets; these control the shape's location within the drawing canvas and its size.
The order of the arguments is: horizontal position (x), vertical position (y), horizontal size (width) and vertical size (height) and they are each separated by a comma.
The horizontal pixels start at 0 at the left hand side of the canvas, so x increases from left to right.
The vertical pixels start at 0 at the top of the canvas, so y increases from top to bottom.
