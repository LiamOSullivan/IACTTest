Colour From Numbers

Different colours can be generated by suppying different numerical arguments to colour functions.
Here, circles and squares are coloured on a grey canvas.

How it works:

Individual colour values range from 0 to 255 (256 values).
When used as a single argument this gives levels of grey from black to white
The canvas is therefore coloured grey using background(127).
Providing three arguments to a colour function sets values for the red, green, and blue mix.
This is called RGB colour
A bright red shape is therefore achieved using fill(255,0,0).
A bright green shape with fill(0,255,0) and so on.
The code examples show how to mix colour in various ways.
