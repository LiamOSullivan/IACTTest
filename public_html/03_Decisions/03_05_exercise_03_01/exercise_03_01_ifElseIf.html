Exercise 03.01 - If Else If

Code which decides how to draw and colour some shapes, based on the states of boolean values.

To Do:

Create two variables to store Boolean values:
useOutlines and
useFillColor
Use these to decide how a pair of shapes of your choosing should be drawn.
Make sure the shapes are always visible with a single if-else-if structure.
If useOutlines is true, the shapes should have a stroke but no fill colour (regardless of useFillColor).
If useOutlines and useFillColor are both false, use BOTH a stroke outline and fill colour anyway.
