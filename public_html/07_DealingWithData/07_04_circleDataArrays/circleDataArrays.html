Circle Data Arrays

Draws 100 circle outlines to the canvas, one by one.

How it Works:


Two arrays are used to store the values for the x- and y-coordinates of the circles.
Random values for the x- and y-positions of the circles are first generated with each stored in one of the arrays.
A count variable is used to keep track of which circle is being drawn. As count is incremented, each new circle appears onscreen.
